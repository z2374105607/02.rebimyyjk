RouteCollection

new Cesium.RouteCollection()

飞行路线集合类。
Example:
var routeCollection = new Cesium.RouteCollection();

Members

readonlyready : Boolean

获取路线集合是否准备就绪，若fpf文件解析完成且成功则返回true。
Default Value: false

readonlyreadyPromise : Boolean

获取该路线集合就绪状态的Promise。

readonlyroutes : RouteCollection

获取路线集合，即从fpf文件解析出来的Route数组，目前只支持一个Route.

Methods

addRoute(route)

向集合添加飞行路线。
Name Type Description
route Route 待添加的飞行路线对象。

fromFile(url)

从fpf文件加载飞行路线。
Name Type Description
url String 指定fpf文件路径。

var routeCollection = new Cesium.RouteCollection();
 routeCollection.fromFile(fpfUrl);
