Cartographic

new Cesium.Cartographic(longitude, latitude, height)

由经度、纬度、高度定义的空间位置。
Name Type Default Description
longitude Number 0.0 optional 以弧度为单位的经度。
latitude Number 0.0 optional 以弧度为单位的纬度。
height Number 0.0 optional 以米为单位的高度。
