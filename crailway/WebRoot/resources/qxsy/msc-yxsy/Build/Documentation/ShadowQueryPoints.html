ShadowQueryPoints

new Cesium.ShadowQueryPoints(scene)

阴影率分析类，指定分析范围、设置阴影类型、设定分析时间段得出阴影率分析结果。
Name Type Description
scene Scene optional 场景对象。
Throws:
  • DeveloperError : 必须指定场景scene。
Example:
//创建阴影率分析
var shadowQueryPoints= new Cesium.ShadowQueryPoints(scene);
