Globe

new Cesium.Globe(ellipsoid)

场景所渲染的球体，包括地形(Globe#terrainProvider)和影像图层(Globe#imageryLayers)。
Name Type Default Description
ellipsoid Ellipsoid Ellipsoid.WGS84 optional 确定地球的大小和形状。

Members

baseColor : Color

当无影像图层可用时，获取或设置地球的颜色。

globeAlpha : Number

获取或设置地球的透明度。 透明度取值范围为0-1；0表示完全透明，1表示完全不透明。
Demo:
  • 地下管线

imageryLayers : ImageryLayerCollection

获取地球上被渲染的影像图层集合。

maximumScreenSpaceError : Number

The maximum screen-space error used to drive level-of-detail refinement. Higher values will provide better performance but lower visual quality.
Default Value: 2

show : Boolean

确定是否显示地球。
Default Value: true

terrainProvider : CesiumTerrainProvider

获取或设置地球的地形服务提供者。.
